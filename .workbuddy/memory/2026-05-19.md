## 头像上传改为图床 API

### 修改内容
- 文件：`user-center.html`
- 函数：`confirmAvatarChange`
- 修改：将头像上传后端从 localStorage base64 改为 `https://img.scdn.io/api/v1.php` 图床 API

### 技术细节
1. Canvas 裁切压缩图片为 200x200 的 Blob
2. 通过 `fetch` POST 到 `https://img.scdn.io/api/v1.php`
3. API 支持 CORS（`Access-Control-Allow-Origin: *`），可直接从前端调用
4. 拿到外链 URL 后存到 Supabase（`update_user_avatar` RPC）和 localStorage
5. Supabase 同步失败不阻断，仍保存到本地

### 注意事项
- `img.scdn.io/api_docs.php` 是文档页，真正 API 端点是 `/api/v1.php`
- 图床限制：5秒内 ≤5次 POST，60秒内 ≤120次
